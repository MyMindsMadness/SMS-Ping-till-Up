# SMS-Ping-till-Up
 This repo uses TextBelt and Ping3 within python to ping an IP address or URL and will send an SMS message/Notification when the device is declared stable.

# Create Virtual Environment 
 py -m venv venv

# Create .env file 
 touch venv/.env

# Add API KEY to .env
 API_KEY='Your TextBelt API key'
